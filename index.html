Sirkulasi Buku (Manual)

Fitur Pinjam dan Kembali adalah proses **manual**.

Sebagai petugas, Anda harus membuka Google Sheet "katalog buku" dan mengubah kolom "Status" secara manual dari "Tersedia" menjadi "Dipinjam" (atau sebaliknya).

Website ini akan otomatis ter-update setelah beberapa menit.

Pengaturan (Manual)

Semua pengaturan dilakukan secara manual:

  • Upload Data: Buka file Google Sheet Anda dan ketik data baru di baris baru.
  • Upload Logo: Upload file logo (misal `logo.png`) ke repository GitHub Anda, menggantikan file yang lama.
